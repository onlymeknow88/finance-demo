Template Dokumen

Gunakan file template kami untuk memastikan format kolom sudah sesuai sebelum melakukan import.

Download Template .csv / .xlsx

Petunjuk Pengisian

  • Kolom Tanggal: Gunakan format internasional YYYY-MM-DD (contoh: 2026-07-05).
  • Kolom Tipe: Tulis Income untuk pemasukan, dan Expense untuk pengeluaran.
  • Kolom Kategori: Sesuai dengan nama kategori terdaftar (Operasional, Marketing, Gaji, dll). Jika tidak dikenal, sistem otomatis memasukkan ke kategori "Lainnya".
  • Kolom Project: Tulis nama project (mis. Project Alpha, Project Beta).
  • Kolom Nominal: Isikan angka saja tanpa format titik/koma desimal ataupun simbol mata uang (contoh: 1500000).
